
Log viewer

installing dependicies global
sudo npm install -g
or local to ./node_modules
npm install

change root of globalsdb var in logs.js

use console 

Usage: node ./app.js [commands] [arguments]

Commands:
   start - start webserver
   addlog - Add log file
   viewlog - view log
   refresh - refresh from file
   list - show list logs
   remove - remove log

or start webserver on 3000 port
node ./app start

system files log can add only console mode
system files on start webserver, start autorefresh
