Overview

Tundra project allows you to quickly create Web applications without any line of server-side code! The Tundra's main idea is to integrate Cloud NoSql Database and client-side framework which provides transparent CRUD-functionality for JS objects.

Since you choose Tundra, you:

  • Don't worry about hosting. All you need is to create new Tundra Project. At this point your site is already online.
  • Don't worry about server code.Just write your business code using only JavaScript. Tundra makes this process pretty simple, fast and fun.
  • Don't worry about storage engine.Our database supports relational paradigm as well as NoSql approach. Tundra provide ability for indexing, validations and transactions.

Quick Start

  1. Click on Projects->Create. Then type desired project name.
  2. Create your project database schema

    At this step you should add tables corresponding to the entities you desire to use in the project.

    Feel free to add needed columns for each table.

    Submit it!

  3. Include following scripts to your HTML-file.

    JQuery:

    <script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.6.4/jquery.min.js"></script>

    Tundra core:

    <script type="text/javascript" src='http://93.88.167.141/public/js/tundra.js'> </script>

    Your models (use concrete link that you will receive after schema submitting!):

    <script type="text/javascript" src='http://93.88.167.141/public/js/models/models1.js'> </script>
  4. Enjoy It!

Note that after schema creation you are able to download auto-generated template of your views. It consist of js-file and html file. Feel free to download and modify these resources to quick start with Tundra.

JavaScript layer guide

Assume you've just created schema with one table Person which has 3 columns: "firstname", "lastname", "age"

Including of auto-generated js-file to your html gives you access to following powerful functions:

  1. Creating new object

    2. First create a JS-object:

    person = new Person();
person.firstname="Alex";
person.lastname="Johns";
person.age=28;

    Another way:

    person = new Person({firstname: "Alex", lastname:"Johns", age:28});

    Now you're able to store it in Tundra database!:

    person.save(function(){
  alert('successfully saved!');
});
  2. Retrieving object
    3. Person.open(1,function(person){ // get person with Id=1
  alert(person.firstname);	// output "Alex";
});
  3. Update object
    4. Person.open(1,function(person){
  person.age = 29;
  person.save(function(){
    alert('Age was succesfully changed to 29');
  });
});
  4. Delete object
    5. Person.deleteId(1,function(){
    alert('Alex left');
  });
});
  5. Find and sort

    6. With Tundra filtering and ordering become so simple!

    Person.where('age','<',28).order("age","desc").getObjects(function(people){
  RedrawPeople(people);
});
  6. Add dynamic behaviour

    7. You may add methods to your objects

    Person.method("fullname",function(){
  return this.firstname + " " + this.lastname;
});

// Use method
Person.open(1,function(person){
  alert(person.fullname());
}
